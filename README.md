# Polydrome

### A palindromic ELF/DICOM polyglot for BGGP6.

An entry for [BGGP6](https://binary.golf/6). A write-up will follow soon-ish...

![Polydrome](./polydrome.png)

### Testing

1. Install build dependencies (tested on Debian 13 Trixie)

> Note: Some Canonical repos do not ship `dcm2img` with the `dcmtk` package.

```
sudo apt install build-essential nasm dcmtk qemu-user feh python3 xxd
```

2. Build the polydrome binary: `make`.
3. Test ELF functionality: `make elf`.
4. Test DICOM functionality: `make dicom`.
